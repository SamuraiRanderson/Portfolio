Neighborhood Viewer

Using the Google Maps, Wikipedia and The New York Times Api's to show relevant information based on a users address query. The user is able to enter a address in the search bar and the application will return Wikipedia information for the queried address, relevant New York Times articles for the area and a Google Streetview image of the address. This was an interesting project as it was my first experience using Apis and realizing the wealth of data that can be accessed, and incorporated, into website to increase to functionality. -Completed 2015
