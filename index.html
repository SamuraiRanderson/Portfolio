Showcasing locations throughout Dallas, TX

Neighborhood Map is a single page application that uses Google Maps Api, and the Instagram Api, to showcase a variety of locations throughout Dallas, TX. The user is able to click on a map marker of a location in downtown Dallas and a custom map marker will display the selected locations contact information, and include the latest Instagram photo tagged with that location. A search bar and list view are available so the user is able to filter through the locations.

Performance Matters!

I was provided an existing website that had a number of optimization, and performance, related issues. The goal was to improve the site's Google Page Speed score, increase the FPS of the animations on the page, and improve the animation resizing times while preserving the sites appearance, animations and functionality. I used a number of optimization techniques to boost performance from the original Page Speed scores of 27/100 to 93/100, and increased performance to 60FPS from the original 25FPS while reducing the animation resizing time from 83.33ms to 0.46ms. -Completed 2015
